# 产品界面与历史记录交互优化

本 PR 将前端从基础录音页面优化为完整的语音输入法工作台。

## 新增能力

- 文本编辑器
  - 展示实时识别和离线识别文本
  - 支持手动编辑
  - 支持复制
  - 支持清空

- 历史记录面板
  - 分页加载历史识别记录
  - 支持刷新
  - 支持复用历史文本到编辑器
  - 支持编辑历史文本
  - 支持删除历史记录

- 产品化界面
  - 展示实时、准确、可控三项能力
  - 保留 StepFun 实时和 Vosk 离线引擎切换
  - 优化移动端布局

## 验证点

- 页面布局正常。
- 录音识别后文本能进入编辑器。
- 文本可以复制和清空。
- 历史记录可以刷新、复用、编辑和删除。
